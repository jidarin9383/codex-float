## Trademark note

Do **not** ship the official OpenAI / Codex logo as this app’s icon or menu bar mark.

- OpenAI and Codex are trademarks of OpenAI.
- A third-party Mac utility should use an **original** mark that only *suggests* coding/agent tooling.
- Safe: `>`, `>_`, chevrons, progress rings, original glass orb.
- Risky: copying Codex product icon, OpenAI logomark, or confusingly similar trade dress.
